No Control

No Control is completely free - no ads, no hidden fees, forever. No Control is a simple virtual controller which allows you to map keyboard keys to an XBOX controller. This work would not be possible without the amazing library made by ViGEm. No Control is detected by most games as a valid controller, so you should have no issues using it. It can also be used in the Steam Remote Play, so that you don't need to share mouse and keyboard inputs.

Installation

Download the following zip folder. Unzip it in any location of your choice. Start the program by running the nocontrol.exe. On your first time opening it, it may request that you install the ViGEm drivers. Don't worry, this is not a virus. It is simply the drivers for the virtual controller. You can uninstall them through the VIGEM_setup.msi located in extras.

Usage

The controller is on by default. You can stop it by pressing the button or with the key F6.

!!! While the controller is working all bound keys will not work! They are mapped to a controller input. If you want to use your keyboard, stop the application with the button or via the hotkey F6.

Remapping Keys

To do this first you need to stop the controller (press the button on the screen or use the F6 shortcut). You should now see a green button on the bottom right. You can remap keys by pressing on the box next to the controller image. The box will turn to "SOME NUMBER: ...". Simply press the new key you want to map to the controller function.
